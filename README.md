# mon.net
.NET Monitoring project with ASP.NET Core and Windows Services.
Client Commandline Arguments:

 - -i: install service
 - -u: uninstall service
 - -s: start service
 - -e: stop service
 
**Example:** 
"Mon.NetClient.exe -i -s" would install and then start the service.
